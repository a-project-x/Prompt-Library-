# 🎯 Prompt Name: The Storyteller Framework

## 🧠 Purpose
Create emotionally-driven brand stories using the Hero's Journey framework.



## 📝 The Prompt
Use the Hero's Journey framework to write a short brand story for [Company]. The founder is the guide, the customer is the hero. Focus on the emotional lowest point before the breakthrough. Tone: Raw, authentic.
