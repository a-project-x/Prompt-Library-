# 🎯 Prompt Name: The High-Conversion MLS

## 🧠 Purpose
Write emotional real estate listing descriptions that focus on lifestyle and sensory experience.



## 📝 The Prompt
Write an MLS description for [Property Details]. Do not use "cozy", "stunning", or "must-see". Start with a sensory experience of living in the house.
