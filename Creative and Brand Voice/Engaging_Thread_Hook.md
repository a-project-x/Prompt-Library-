The Engaging Thread Hook

Overview

A prompt system designed to generate attention-grabbing opening hooks for X (Twitter) threads.

The framework focuses on creating curiosity and credibility without relying on clickbait.

Core Prompt

Create a hook for an X thread about:

[Topic]

Requirements:

- Start with a bold metric, observation, or insight
- Create curiosity without misleading readers
- Avoid generic statements
- Keep under 25 words
- Encourage continued reading
