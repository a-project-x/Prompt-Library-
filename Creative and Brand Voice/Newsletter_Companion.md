The Newsletter Companion

Overview

A newsletter writing system designed to create personal, conversational introductions that feel like a message from a trusted friend rather than a marketing broadcast.

The framework focuses on connection before information.

Core Prompt

Write a weekly newsletter introduction about:

[Topic]

Requirements:

- Write as if speaking to one friend
- Use a conversational tone
- Avoid corporate language
- Include a personal observation or reflection
- Create curiosity for the rest of the newsletter
- Keep the introduction natural and concise
