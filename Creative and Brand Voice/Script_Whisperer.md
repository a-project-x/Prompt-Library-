The Script Whisperer

Overview

A short-form video scripting system designed to maximize viewer retention during the critical first few seconds of a video.

The framework combines visual interruption with verbal curiosity.

Core Prompt

Write the first 5 seconds of a YouTube Short or Reel about:

[Topic]

Requirements:

- Include a visual instruction
- Include a spoken line
- Create curiosity immediately
- Avoid generic openings
- Keep the hook concise
- Encourage viewers to continue watching
